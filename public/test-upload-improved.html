🔧 Chrome DevTools 使用指南

1. Network 标签页监控

  • 查看请求头：点击请求 → Headers 标签
  • 查看请求体：点击请求 → Payload 标签
  • 查看响应：点击请求 → Response/Preview 标签
  • 查看时间：点击请求 → Timing 标签

2. Console 错误监控

  • 查看 JavaScript 错误
  • 查看网络请求错误
  • 查看服务器日志信息

3. 关键指标检查

  • Content-Type: 应为 multipart/form-data
  • Request Size: 文件大小 + 边界信息
  • Response Status: 200 (成功) 或 4xx/5xx (错误)
  • Response Time: 上传和处理时间
